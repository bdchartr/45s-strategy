# wkapp-45-strategy

Research project to develop strong AI strategies for the 45s card game via self-play reinforcement learning on Apple Silicon.

Target architecture: AlphaZero-lite. Rust game engine, Python orchestration, MLX or PyTorch MPS neural networks, actor-learner parallelism.

Sibling project: [bdchartr/45s-2026](https://github.com/bdchartr/45s-2026) — the production 45s web app and source of truth for game rules.

See `GOALS.md` for the full plan and `TODO.md` for current status.
